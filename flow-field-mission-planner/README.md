# basic-A-star-mission-planner

this repo contains a basic mission planning engine that utilizes A* search algorithm to find the optimal flight path from A to B through threats and obstacles. it's basically the primitive version of something you would use to map out and plan the trajectory of your drone from point a to b. I am working on the advanced versions but i had zero clue about searching algorithms so here's my little experiment to figure out how A* actually works.
